Chess Question

We are going to build a chess game. These questions are designed to specifically test your understanding of inheritance, scope, embedded loops, and if you can think in code. You'll be asked to use psuedo code, so the language isn't important.

At a base level, chess is a game with two players, a set of pieces that have prescribed moves, a board with positions that can be represented by X and Y coordinates.

NOTE: Answers could potentially build from previous answers.

POINTS SYSTEM: Points are described below with a number starting with the worst answer to the best.
"Ways to help" questions are to be asked if candidate is stuck. Wait for them to answer, only if they can't, give them the answer that goes with that question.
NOTE: If you provide that answer, they will be penalized in the points system.

    What kind of operations do you think each piece would need to have? "How a piece moves method", "available moves method" and "XY coordinates"
